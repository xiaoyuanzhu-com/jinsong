Signal quality audit · SIG-1

The 57% finding.

The Claude Code connector consumes the happy-path event shape only. It never reads tool_result.is_error — the canonical failure marker.

sessions with silent tool failures
444 / 776  57.2%
api_error retry events silently dropped
326 events · 62 sessions
tool_call_end status distribution today
success 29,226 · timeout 705 · failure 0

Consequence. rel_error_rate, rel_hidden_retries, retry_count_total, errors_per_session are uniformly zero across the corpus. tool_success_rate = 1.0 reported for 444 sessions that actually contain 2,846 tool failures.

Fix scope. Small. Tracker already handles the events — bottleneck is purely extraction. Unlocks 6+ metrics with one connector change.

next rolls into RM-3 as SIG-3/4/5/6
